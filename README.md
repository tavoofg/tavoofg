## 🙋‍♂️ Sobre mí

Soy **Ingeniero de Sistemas** egresado de la **Universidad Nacional de Cañete (UNDC)**, con más de 1.5 años de experiencia en **Quality Assurance y automatización de pruebas**.

Me especializo en testing de frontend y backend (Selenium, Playwright, Karate), diseño de casos de prueba BDD con Gherkin, pruebas de rendimiento con JMeter y K6, e integración continua con GitHub Actions y Azure DevOps. También disfruto desarrollar soluciones web con IA aplicada.

- 🏙️ **Lima, Perú**
- 🎓 **Décimo superior** — Ingeniería de Sistemas, UNDC
- 🔍 Especializado en **QA Automation** (Frontend & Backend)
- 🤖 Apasionado por la IA aplicada y el desarrollo web
- 🚀 Siempre buscando automatizar lo que se pueda automatizar
